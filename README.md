# CollabDesk 

A simple team collaboration tool.

---

## About

CollabDesk is a full-stack project built to help teams manage their work in a simple and visual way. You can create projects, add tasks, assign them to team members, and track progress using a Kanban-style board.

This project is mainly built for learning purposes, but it’s structured in a way that can be extended into a real-world application.

---

## Features

* User signup & login
* Create and manage projects
* Add, edit, and delete tasks
* Assign tasks to team members
* Task status (To Do, In Progress, Done)
* Simple dashboard to track progress

## Getting Started

### Clone the repo

```
git clone https://github.com/
cd collabdesk
```

---

### Backend setup

```
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload
```

---

### Frontend setup

```
cd frontend
npm install
npm start
```
